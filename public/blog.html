1. What is the difference between Props and State?

Props : is used to pass data form the parent component to child component. This data can only be modified in the parent component not in the child component.
State: in opposite State is used to manage and modify data in the same component. state itself an array and holds the current sate of component and modify the data.

2. How useState work?

useState: is React hook uses to hold current state and modify the data.useState works in two steps
initially set a initial value of the data
secondly modifies or updates the data by function which is the second element of the useState array

3. What is the work of useEffect except data loading?

Except data loading useEffect can be used as follows:
  • When a component is unmounted, useEffect can be used to clean up after it.
  • When a component updates, useEffect can be used to start animations or transitions.
  • After a component has rendered, useEffect can be used to update the DOM.

4. How does React Work?

React is a JavaScript library used to create user interfaces. It makes effective use of a virtual DOM (Document Object Model) to refresh the UI in response to changes in user interactions. To manage state and data, it also employs a component-based architecture and a unidirectional data flow.
